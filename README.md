Sistema de Precificação para Confeitaria (Beta)

Aplicação desktop desenvolvida em Python para automação de precificação e controle de estoque de produtos de confeitaria.

Funcionalidades
Cadastro de produtos
Controle de estoque
Cálculo automático de preço
Cálculo de lucro
Interface gráfica
Importação de planilhas Excel
Banco de dados SQLite
Tecnologias utilizadas
Python
CustomTkinter
SQLite
OpenPyXL
Objetivo do projeto

Projeto pessoal criado para automatizar o processo de precificação e organização de produtos de confeitaria.

Status

Versão Beta — Em desenvolvimento.
